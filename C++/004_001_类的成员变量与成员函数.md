# 类的成员变量与成员函数

1.在类体中直接定义函数，不需要在函数名前面添加类名

```c++
class Student{
public:
    //成员变量
    char *name;
    int age;
    float score;

    //成员函数
    void say(){
        cout<<name<<"的年龄是"<<age<<"，成绩是"<<score<<endl;
    }
};
```

2.在类体中声明函数，将函数定义放在类体外

2.在类体中声明函数，将函数定义放在类体外

```c++
class Student{
public:
    //成员变量
    char *name;
    int age;
    float score;

    //成员函数
    void say();  //函数声明
};

//函数定义
void Student::say(){
    cout<<name<<"的年龄是"<<age<<"，成绩是"<<score<<endl;
}
```
