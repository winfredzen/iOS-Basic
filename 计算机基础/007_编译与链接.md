# 编译与链接

内容来自：《程序员的自我修养》



IDE一般将编译和链接的过程一步完成，称为构建**Build**

如下简答的C程序hello.c：

```c
#include <stdio.h>
int main()
{
    printf("Hello World\n");
    return 0;
}
```

GCC编译时

```shell
gcc hello.c
./a.out
Hello World
```

事实上上面的步骤可以分为4个步骤

+ 预处理（Prepressing）
+ 编译（Compilation）
+ 汇编（Assembly）
+ 链接（Linking）
